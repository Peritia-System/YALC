# YALC — Yet Another Linux Course

Welcome to **YALC (Yet Another Linux Course)**.
This is a practical learning path designed to help you understand **Linux**, **IT fundamentals**, **web development**, and **homelab basics** through hands-on exploration.

Whether you are new to Linux or want to build a stronger foundation, this course offers a structured way to learn step by step.

## Note 

This is a Work in progress so feel free to contribute and please dont rely on my information soley. I will Reference Guides and Videos to check out as best as I can

---
